# Airline Management System (AMS) - Admin Frontend Project Summary

## Project Overview

This is an Angular 17 standalone component based Admin Frontend for an Airline Management System (AMS).

Backend:

* Spring Boot
* H2 Database
* REST APIs running on localhost:8080

Frontend:

* Angular 17
* Standalone Components
* Angular Router
* HttpClient
* Glassmorphism UI redesign completed

## Initial Issue

Frontend was showing a blank white screen.

Root cause:

* app.component.html was empty
* Angular rendered only:

<app-root></app-root>

Fix:
app.component.html

<router-outlet></router-outlet>

app.component.ts

imports: [RouterOutlet]

After this routing started working.

Project uses Angular Standalone Components.

Do NOT generate NgModule-based code.

Always use:

standalone: true

imports: []

Angular version:

17.x

Keep all future pages consistent with existing glassmorphism airline operations theme.
